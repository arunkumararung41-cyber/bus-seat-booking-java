# 🚌 Bus Seat Allocation System (Java)

## 📌 Overview
This project simulates a simple bus booking system using Java ArrayList.

## 🚀 Features
- 3 buses: Mumbai, Hyderabad, Chennai
- Each bus has only 3 seats
- Users assigned based on destination
- Prevents overbooking

## 💡 Tech Used
- Java
- ArrayList
- OOP Concepts
